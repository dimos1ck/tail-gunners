# TAILGUNNERS
1. Концепт (The Elevator Pitch)
Tailgunners — это динамичный сетевой 2D-шутер, где главные герои — боевые коты, использующие свои хвосты как турели.

3. Ключевые механики (Core Gameplay)

    Хвост-манипулятор: Оружие закреплено на кончике хвоста. Это дает уникальный сектор обстрела: кот может бежать вперед, а стрелять назад или вверх под углом, недоступным обычным гуманоидным персонажам.
    Сетевой PvP: Бои в формате Deathmatch
    Паркур: Двойные прыжки, карабканье по стенам и подкаты — коты должны быть мобильными.
    Система кастомизации: Основной акцент на головных уборах (шлемы, фуражки, очки).

4. Технический стек (Unity Implementation)

    Physics 2D: Использование Rigidbody2D для перемещения и BoxCollider2D для окружения.
    Tail System (IK): Реализация хвоста через 2D Inverse Kinematics (IK). Хвост должен плавно следовать за прицелом мышки, создавая ощущение живого придатка.
    Networking: Для начала стоит смотреть в сторону Photon Bolt или Mirror (проще для старта 2D-шутера).
    Shooting: Система снарядов (Projectiles) для пушек с разной скоростью полета и отдачей, которая физически толкает кота назад.

5. Визуальный стиль

    Персонажи: Суровые коты разных пород.
    Сеттинг: Дизельпанк / Авиабазы. Стальные платформы, ангары, летящие самолеты-платформы.
    Интерфейс: В стиле приборной панели старого истребителя.
